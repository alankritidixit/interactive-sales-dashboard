# Sales Dashboard in Excel

## Overview

This project showcases an interactive Sales Dashboard built entirely in
Microsoft Excel using a dataset of approximately 10,000 records. The
goal of this dashboard is to transform raw sales data into actionable
insights through effective data visualization and analysis techniques.

The dashboard enables users to explore sales performance, identify
trends, and make data-driven decisions using interactive elements such
as slicers and pivot-based visuals.

## Project Highlights

## Key Features

-   Sales by Category analysis (Furniture, Office Supplies, Technology)
-   Profit trend visualization over multiple years
-   State-wise sales distribution map
-   Customer count comparison across years
-   Top 5 profit-generating customers
-   Monthly sales trend analysis
-   Interactive filtering using slicers (Category, Year)

## How to Use

1.  Download or clone this repository
2.  Open `Sales_Dashboard.xlsx` in Microsoft Excel
3.  Use slicers to filter by:
    -   Category
    -   Year
4.  Explore insights through interactive charts
